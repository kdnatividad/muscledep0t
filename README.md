## GET STARTED

```bash
# Clone the repo
git clone https://github.com/kdnatividad/muscledep0t.git

# Enter the project
cd muscledep0t

# Install dependencies
npm install

# Run locally
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and start training.
